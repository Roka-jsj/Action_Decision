# 전략 성패의 통일 이론 (R33, 2026-07-08) — 전 실측 종합

근거: LB 실측 40여 건(R1~R32), 오프라인 실측 수십 건, 상수 프로브 3발 역산. 상세 로그 = DEBATE.md, experiments_master.csv.
용도: **모든 신규 제안은 아래 M1~M5 필터를 통과해야 채택** — codex 브리핑(R33+)에 이 표를 인라인하고, 제안마다 "어느 사망 메커니즘에 왜 안 걸리는가"를 명시할 것.

## 1. 실패의 5대 사망 메커니즘

### M1. 분포이동 하의 승자의 저주 (train-분포 선택편향)
**사례**: v9 rich(-0.011) / histdrop(-0.011) / v8 [GEN]재배치(fold 전폴드 +0.0142 → LB -0.0023) / sim-only(-0.009) / 메타v7(-0.008)
**이론**: 가설을 D_train 성능으로 선택하면 선택된 가설은 "진짜 신호 + train 특이 지름길"의 합이 최대인 것이다. 히든이 OOD(홀드아웃→train near-dup 이득 +0.023 vs 히든→train +0.0005, 46배차 실측)면 지름길 성분의 전이는 0 이하 — **관측 이득이 클수록 지름길 비중이 클 확률이 높아 기대 반전폭도 커진다.** v8이 결정적 사례: 5-fold 전부에서 견고했던 +0.0142가 실은 [GEN]을 coarse-prior에서 GEN×prompt 상호작용 지름길로 전환한 것(R19 층화진단: hist=0 절단 0% 구간이 최대 이득 → 절단복원 가설 기각).
**따름정리**: 클린 fold 게이트는 필요조건일 뿐 충분조건 아님(오염은 막지만 지름길은 못 거름). 최종심은 항상 LB.

### M2. 암기 오염 (FULL모델 홀드아웃 평가 무효)
**사례**: lgb8(OOF +0.0015→LB -0.0024) / em075(holdout +0.0020→-0.0009) / tri_sb(holdout +0.0063→-0.0017) / dist3(증류는 암기가 옅어 역핸디캡으로 판정 왜곡)
**이론**: FULL 학습은 홀드아웃 행 자체를 봤다. holdout Δ = Δ일반화 + Δ암기이며, 체크포인트에서 추가 학습(FT/증류/스택)할수록 Δ암기가 지배. nnQ1(low near-dup) 슬라이스 방어도 무력 — near-dup이 없어도 행 자체를 암기했기 때문. 3연속 실증으로 **FULL-holdout 델타 기반 의사결정 공식 금지**(R31~R32).

### M3. 목적함수 기하 불일치 (macro-F1 ≠ 카운트 일치)
**사례**: blind EM em075(-0.0009) / 정답앵커 count-matching(오프라인 -0.0003~-0.0009) / dual-bias(-0.0009)
**이론**: macro-F1 최적해는 클래스별 precision/recall 교환비의 균형점이고, 저정밀 클래스는 **의도적으로 유병률 이상 과다예측**하는 것이 최적(list_directory: 최적 예측률 9.3% vs 진실 6.5%, precision 0.384). 예측분포를 진실분포에 맞추는 모든 보정(EM/count-matching/prior-matching)은 이 최적 교환비를 되돌리는 **다른 목적함수**다. 프로브로 정답 prior를 알아도 마찬가지(실측으로 증명). dual-bias는 여기에 au 4,622행 소표본 과적합이 중첩.

### M4. 신호 포화 + 알레아토릭 바닥
**사례**: reranker/ngram(탐색4class +0.0007뿐) / 메타 corrector(로짓 조건부 잔여 0) / specialist 4-way(표적셋 spec 0.360 vs base 0.375)
**이론**: 오류가 몰린 저마진 구간(base acc 0.375)은 용량 부족이 아니라 본질적 모호성이 지배 — near-dup 이웃 간 라벨일치 69%(같은 상황에서 실제로 다른 행동), 탐색행 61%는 텍스트에 판별 신호 부재(R6). 같은 데이터로 학습한 어떤 전문가·보조모델도 이 바닥을 못 뚫는다. large는 텍스트·메타 프록시를 이미 흡수 완료(R7·R9).

### M5. 약멤버 희석 + 가중공간 비볼록
**사례**: klue(-0.014) / base 스택 lgb8(-0.0024) / soup V자 붕괴(50:50 최저) / SWA(-0.006)
**이론**: 확률 평균의 이득은 오차 탈상관이 전제 — 약멤버는 자기 오차를 주입해 강멤버를 희석(강-강만 유효, str2 실증). 가중치 평균(soup)은 시드가 다르면 loss 분지가 달라 중간점이 계곡. SWA는 6→8ep 미수렴 구간 평균이라 후퇴.

## 2. 성공의 4대 생존 메커니즘

### S1. 메트릭 구조의 직접 최적화 — bias +0.0058 (최대 단일 이득)
14파라미터(저복잡도) × 64k OOF(대표본) × 히든 prior≈train(프로브 확증) → 과적합 여지와 분포 의존이 동시에 최소. 점수를 바꾸는 건 모델이 아니라 **결정규칙**일 수 있다.

### S2. 강-강 분산감소 + 조건부 예산배분 — +0.002~4
largeonly .78051 → cond2 .78162 → tri_cond .78266. 동일 강도·다른 시드/직렬화의 오차 탈상관. 이득이 저마진 행에 집중 → 27%만 재추론해 full 이득 79% 회수(600s 캡 안의 유일한 앙상블 구조).

### S3. 히든에 실존하는 신호의 수호 — [GEN]/World C
no-GEN 제거 -0.047이 역설적 최대 발견: 히든에 id 표식 실존, [GEN]이 FULL부스트 +0.041의 핵심. "train 특이처럼 보이는 신호"도 히든에 실존하면 최대 자산 — 단 그 위치/형태를 바꾸는 순간(v8) M1 지름길로 변질.

### S4. 측정 > 추정, 사전등록 > 사후해석
상수 프로브 3발(노이즈 0 역산) = prior축 영구 종결. 클린 fold0 프로브 = specialist를 제출 0발로 폐쇄. 사전등록 밴드(R31) = 실패도 즉시 해석 가능한 정보로 전환. **제출은 도박이 아니라 실험 설계다.**

## 3. 대회의 구조적 정체 (종합)
1. 히든 = 같은 생성기 · **다른 support**(near-dup 희박) · **같은 prior**(프로브) · **id 표식 있음**(World C) · 세션당 ~1관측.
2. 현 파이프라인 post-hoc 상한 ≈ 0.7833~0.7837(R18), 은행 0.78266 — 그라인딩의 이론적 몫 +0.0007~0.0011.
3. 0.79~0.80 팀과의 갭은 측정된 어떤 축으로도 설명 불가. 잔여 가설: (a)이종 사전학습 백본의 오차 구조 (b)외부데이터/재생성 (c)public⊊private면 상위권 일부는 선택편향 착시(35발 좌표상승만으로 +0.0067~0.008 부풀림 가능, R18 — 미증명).
4. ∴ 발수는 S1·S2 인근에 집중, 신규 축은 M1~M5 필터 통과분만 소량, **최종 선정은 선택편향 방어 규칙**(은행1=저탐색 tri_cond + 은행2=그라인딩 최고 — 두 세계 모두 후회 최소화).

## 5. R33 codex 적대검증 반영 (승인 + 수정 2건)
### M6. public 다중비교 승자의저주 (신설)
N발 중 public 최고 선택의 기대 낙관편향 ≈ 2.1σ_public(N=42). 채택문턱 +0.0005가 유효하려면 σ_public<0.00024 필요(비현실). **규칙: 변형 탐색(학습·조립)은 무제한, public 제출은 10~15발 내에서 사전등록 가설만. 은행 선정은 "사전등록(OOF/이론) 최고 + LB는 확인/거부권"** — LB-max 직접 선택 금지.
### S5. 희소 고정밀 패치 (신설)
retr_v6 largeonly +0.00045 = gate 0.49% × per-row +9.2%p — 좁은 게이트·고정밀 개입은 M1 노출을 최소화하면서 실신호를 잡는다. S4(안전성)와 별개의 이득 메커니즘. 단 게이트 확장(retr_mid 18%)은 즉시 M1 노출.
### M4 완화
"저마진=전부 알레아토릭" → "알레아토릭 비중이 높다". 잔여 추정분산 실재(강-강 +0.002~4 중 미회수 21% ≈ +0.0004~0.0008). R18 상한의 +0.0007은 그럴듯, +0.0011은 선택노이즈와 구분 불가.
### 이중은행 확정 (p(public≈private)=0.55 가정)
은행1=tri_cond(저탐색) + 은행2=**사전등록 최고**(public LB는 거부권만). 기대후회 A+B ≈ 0.0002~3 < A+A'(0.0005) < B+B'(0.0008).
### mdeberta 게이트 (락)
residual r=1[y=c]−p, tri 저마진 27%에서 ρ_low 계산. **통과 ρ<0.82 / 강통과 ρ<0.75 / ρ≥0.90 은행 영구금지**. fold0 > 기준−0.0015. 기본 5% 혼합(OOF Δ>+0.00025), 10%는 강통과+fold0손실≤0.0005일 때만.

## 6. R39 삼각측량 교정 (aubias 반전 사후분석 — codex + 독립 Claude 레드팀)
### M2 확장 — 동일멤버 델타도 무효
"같은 멤버니 암기 상쇄"는 무근거 발명이었다. 델타 연산자는 평가분포의 마진 기하에 비선형 의존 — 암기로 뾰족해진 분포 위의 bias 섭동은 "정답 상관된 값싼 뒤집기"를 수확한다. 실증: aubias holdout +0.00286 vs 클린 OOF **+0.00002**(143배 괴리), em075가 1호 반증(당시 M3로만 분류한 것이 오류). **규칙: 클린 분포(OOF)에서 측정 가능한 델타는 클린 측정만 유효.**
### M3 재정의 — 히든 분포 가정 주입 금지
"카운트 매칭"보다 넓게: **히든 분포에 대한 어떤 가정이든 결정규칙에 주입하면 사망** — dual-bias/em075/damped/aubias 4전4패. 성공한 유일한 bias(+0.0058)는 가정 무개입·자연 train분포·64k 대표본.
### M6 확장 — 추정기 선택도 다중비교
기대값 산정에 쓸 추정기를 낙관 쪽으로 갈아타는 것(R37 클린시뮬 +0.00028 → R38 오염 홀드아웃 +0.0017)도 승자의저주. 구 등록치와 괴리나면 해명 필수.
